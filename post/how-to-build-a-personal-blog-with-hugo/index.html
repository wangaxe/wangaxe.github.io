How to Build a Personal Blog With Hugo

2020年7月2日
| 阅读

记录搭博客的全过程 (坑)

Install

Windows

按照 hugo 官网的指南，使用 Scoop 安装。体验相当不错，几乎和 linux 平台 感觉差不多。

1
2
3
4

# 安装 Scoop
Invoke-Expression (New-Object System.Net.WebClient).DownloadString('https://get.scoop.sh')
# 使用 Scoop 安装 hugo
scoop install hugo-extended

Init

  • 初始化博客

    1
2
3

    hugo new site <path>
cd <path>
ls # 是的，powershell可以用 ls :b

  • 安装主题

    在 themes 里选一个喜欢的就好, git submodule add 需要先 git init

    1
2
3
4
5

    git submodule add https://github.com/heyeshuang/hugo-theme-tokiwa.git themes/hugo-theme-tokiwa themes/<theme-path>
  
vim config.toml
# 在 config.toml 文件中指定 theme
theme = "<theme-path>"

  • 本地浏览

    1

    hugo server -t <theme-path> --buildDrafts

  • 部署到github.io

    1

    hugo --theme=<theme-path> --baseUrl="https://<git_user_name>.github.io" --buildDrafts
